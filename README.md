# Iran Crisis Trade Dashboard

Static dashboard for scenario-driven equity ideas around:

- extended Iran conflict
- Strait of Hormuz disruption
- repeated strikes on Middle East oil and LNG infrastructure
- structural LNG shortfall even if the war de-escalates early

## Files

- `index.html` — dashboard shell and styles
- `iran-dashboard.js` — filtering, sorting, and detail view logic
- `iran-dashboard-data.js` — curated trade ideas, scores, and source links

## Open locally

Open `index.html` in a browser.
